ffmpeg 简单用法

最简单的命令如下:

-i 后面是输入文件名
-vcodec 后面是编码格式,h264最佳,如果是要插入到PPT的视频,选择wmv1,wmv2,基本万无一失;

附加选项:
-r 指定帧率
-s 指定分辨率
-b 指定比特率

音频参数:
-acodec 指定音频编码
-ab 指定音频比特率
-ac 指定声道数
