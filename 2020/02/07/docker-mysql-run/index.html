参数说明:

  • -p 3306:3306 映射容器服务的3306端口到宿主机的3306端口,外部主机可以直接通过 宿主机IP: 3306 访问到 MySQL 的服务
  • MYSQL_ROOT_PASSWORD=123456 设置MySQL用户的密码
  • TZ=Asia/Shanghai 设置时区
  • -v /data/mysql-data:/var/lib/mysql 挂载文件存储目录
